## üöÄ Getting Started

> **Prerequisites:**  
> - Node.js (latest LTS)
> - Expo CLI (if using Expo)
> - OpenAI API Key

1. **Clone the repository** (if available in future).


2. **Install dependencies:**
```bash
npm install
```

3. **Configure your OpenAI API key** in the app config.
4. **Start the app:**
```bash
npm start
```


5. **Scan the QR code** with the Expo Go app to run on your device.
